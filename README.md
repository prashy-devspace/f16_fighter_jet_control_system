# Asymmetric Control System Implementation for F-16 Fighter Aircraft

## Overview

This project presents the implementation and simulation of an **Asymmetric Flight Control System for the F-16 Fighter Aircraft** using a **Hybrid Sliding Mode Neural Network (HSMNN) Controller**. The controller is designed to maintain aircraft stability and tracking performance under asymmetric flight conditions caused by control surface failures, structural damage, payload imbalance, or external disturbances.

The system combines the robustness of **Sliding Mode Control (SMC)** with the adaptive learning capability of **Artificial Neural Networks (ANNs)** to achieve improved nonlinear control performance and disturbance rejection.

The complete aircraft model and controller implementation are developed and simulated in **MATLAB/Simulink**.

## System Architecture

The implemented control framework consists of the following major components:

### Aircraft Dynamics Model

The F-16 aircraft model represents:

* Longitudinal dynamics
* Lateral-directional dynamics
* Aerodynamic forces and moments
* Control surface effects
* External disturbances

### Sliding Mode Controller

The Sliding Mode Controller provides:

* Robust tracking performance
* Resistance to model uncertainties
* Fast convergence to desired states
* Disturbance rejection capability

### Neural Network Compensation Layer

The neural network module is responsible for:

* Learning nonlinear aircraft behavior
* Approximating unknown system dynamics
* Reducing chattering effects
* Improving controller adaptability

### Hybrid Control Strategy

The final control signal is generated by combining:

```math
u = u_{SMC} + u_{NN}
```

Where:

* (u_{SMC}) = Sliding Mode Control output
* (u_{NN}) = Neural Network compensation output

This hybrid approach enables robust control while maintaining adaptive behavior in uncertain operating conditions.
